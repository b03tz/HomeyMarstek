Marstek Energy for Homey

Control and monitor your Marstek Venus battery systems (Venus A, C, D, E) directly from Homey over your local network using the Marstek Open API.

Features:
- Auto-discover Marstek devices on your network or add manually by IP
- Real-time monitoring: battery SOC, grid/solar/battery power, temperature, voltage, current
- Energy tracking: grid import/export, solar generation (kWh)
- Control operating modes: Auto, AI, Manual, Passive, UPS
- Set passive charge/discharge power via flows
- Configure depth of discharge (DOD) via flows
- LED panel control
- Full Homey Flow support with triggers, conditions, and actions

Requirements:
- Marstek device connected to your local network
- Open API enabled via the Marstek mobile app (Settings > Open API)
- Default UDP port: 30000
